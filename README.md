# 💧 Daily Water Tracker

## 🌱 Visão Geral
Este projeto permite acompanhar o hábito diário de hidratação através de um gráfico no Pixela. O utilizador interage com um menu simples no terminal, escolhendo entre registar a quantidade de água ingerida no dia ou remover o registo existente. O sistema comunica diretamente com a API do Pixela para gerir utilizadores, gráficos e entradas diárias, garantindo um acompanhamento consistente e visual do progresso.

---

## 🔎 Funcionalidades
- Permite inserir a quantidade de água consumida no próprio dia.
- Caso já exista um registo para o dia, este é atualizado sem necessidade de passos adicionais.
- Possibilidade de remover o registo correspondente ao dia atual.
- Interface simples e intuitiva que facilita a navegação e a escolha das ações.
- Comunicação direta com o serviço para criação de utilizadores, configuração de gráficos e gestão de pixels.

---

## 🧠 Funcionamento
O utilizador escolhe uma das opções apresentadas no menu:

1. Criar ou atualizar o registo do dia  
2. Apagar o registo do dia  
3. Sair da aplicação  

Após a escolha, o sistema recolhe os dados necessários através do terminal e envia-os para o Pixela, mantendo o gráfico atualizado com o progresso diário.

---

## 🎯 Objetivo
O propósito deste projeto é incentivar a consistência no hábito de beber água, oferecendo uma forma simples e visual de acompanhar o progresso ao longo do tempo. A abordagem minimalista torna-o ideal para rotinas diárias e para quem procura manter hábitos saudáveis de forma organizada.
